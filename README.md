# 🚀 CanteenGO - MVP (Prototipe)
**Tugas UAS Technopreneurship 2025 Genap (C) - ITPLN**
**Kelompok SAMTECH**

CanteenGO adalah ekosistem pemesanan makanan kantin berbasis *Pre-Order* dan *Self-Service* untuk memangkas antrean mahasiswa menggunakan algoritma First-In, First-Out (FIFO).

## 💻 Cara Menjalankan Source Code di Localhost
Jika Bapak/Ibu Dosen ingin menguji langsung source code ini:
1. Pastikan **XAMPP** (Apache & MySQL) sudah terinstal dan berjalan.
2. Clone/Download repository ini, simpan di dalam folder `C:\xampp\htdocs\canteengo`.
3. Buka browser dan akses `localhost/phpmyadmin/`. Buat database baru bernama `canteengo_db`.
4. Import file `canteengo_db.sql` yang ada di repository ini ke dalam database tersebut.
5. Akses prototipe melalui browser: `localhost/canteengo/index.php`.
